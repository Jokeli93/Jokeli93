## 🚀 Embedded Projects
### 🔹 STM32 SPI Driver (Register-Level)
Implementation of a bare-metal SPI driver for STM32F407 using direct register access.

#### Features
- SPI peripheral initialization
- Master configuration
- Register-level communication
- Data transmission between microcontrollers

#### Test Setup
- STM32F407 (Master)
- Arduino Uno (Slave)

The STM32 transmits data via SPI which is successfully received by the Arduino.

🔗 Repository:
👉 [View Project]([https://github.com/Jokeli93/stm32_LED_toggle](https://github.com/Jokeli93/stm32f407-baremetal-drivers))
